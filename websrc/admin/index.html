Welcome to MyFileServ

MyFileServ is a Flask-based web application designed to provide users within a local area network (LAN) with a simple and secure solution for file storage and management. With MyFileServ, users can create accounts and upload files of any type and size to their personal storage space, accessible only to them. Additionally, there is a shared space where users can collaborate by sharing files with others in the network. The app offers an intuitive user interface and robust security measures to ensure data privacy and protection. MyFileServ is ideal for households, small businesses, or any environment where users need to efficiently manage and access files within a local network.
